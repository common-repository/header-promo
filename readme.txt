== Description ==

##Best Black Friday, Cyber Monday or any other promo bar is here##
Header Promo is a lightweight plugin, which enables you to display promotional information in the header or footer of a WordPress website. You can use this plugin to show Black Friday, Cyber Monday or other offers.
This promotional bar will help you get more attention from visitors.

= Features =
* Display promo message and a countdown timer in the header and footer.
* Change colors and background for the promo and countdown.
* Customize the button with the normal and hover color and background as well as the border radius.
* User-friendly interface

= How to use =
- After installation, you can see a sidebar menu in the dashboard called "Header Promo"
- Set up the promo and countdown time from there.
- Enjoy!

* For installation help click on Installation Tab
